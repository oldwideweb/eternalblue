# eternalblue

Un site web avec des exemples de CSS vus aux cours de web B2.

Ce site est destiné à s’agrandir, vous pourrez y piocher des inspirations et naviguer de page en page.

Contrairement aux sites créés en classe, je passe un peu plus de temps sur ces pages-ci pour mieux démontrer à quoi peut ressembler un site finalisé, même avec du code simple.

Vous pouvez vous en servir comme *référence*, pas comme template.
